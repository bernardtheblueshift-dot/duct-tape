**1. Create `backend/app/api/v1/calendar.py`:**

Router prefix: `/api/v1/calendar`, tags: `["calendar"]`

**Endpoint: GET /api/v1/calendar/events**
- Query params: `start: datetime`, `end: datetime` (both required)
- Auth: `Depends(get_current_tenant)`, `Depends(require_active)`
- Max range validation: if `(end - start).days > 365` raise HTTPException 400 with detail `f"Date range too large (max 365 days)"`
- Ensure timezone-aware: if `start.tzinfo is None: start = start.replace(tzinfo=timezone.utc)` (same for end)
- Build events list using batch query pattern (NOT N+1):

Step 1 — Query jobs in range:
```python
jobs_result = await db.execute(
    select(Job).where(
        Job.scheduled_start.is_not(None),
        Job.scheduled_end.is_not(None),
        Job.scheduled_start < end,
        Job.scheduled_end > start,
    )
)
jobs = list(jobs_result.scalars().all())
job_ids = [j.id for j in jobs]
```

Step 2 — Batch fetch crew assignments + crew names:
```python
crew_result = await db.execute(
    select(CrewAssignment, User.email)
    .join(CrewProfile, CrewAssignment.crew_id == CrewProfile.id)
    .join(User, CrewProfile.user_id == User.id)
    .where(CrewAssignment.job_id.in_(job_ids))
)
# crew_result.all() returns list of (CrewAssignment, email) tuples
```

Step 3 — Batch fetch equipment assignments + equipment names:
```python
equip_result = await db.execute(
    select(EquipmentAssignment, Equipment.name)
    .join(Equipment, EquipmentAssignment.equipment_id == Equipment.id)
    .where(EquipmentAssignment.job_id.in_(job_ids))
)
```

Step 4 — Build CalendarEvent list:
- For each job: `CalendarEvent(id=job.id, event_type="job", title=job.title, start=job.scheduled_start, end=job.scheduled_end, color=JOB_STATE_COLORS[job.state.value], status=job.state.value)`
- For each crew assignment: `CalendarEvent(id=ca.id, event_type="crew_assignment", title=f"{ca.role} - {job_title}" if ca.role else job_title, start=job_start, end=job_end, color=JOB_STATE_COLORS[job_state], status=ca.status.value, job_id=ca.job_id, resource_id=ca.crew_id, resource_name=email, job_title=job_title, role=ca.role)`
  - Look up job from a `jobs_dict = {j.id: j for j in jobs}` map
- For each equipment assignment: similar pattern with `event_type="equipment_assignment"`, `resource_name=equipment_name`

Return `CalendarEventsResponse(events=events, count=len(events))`
