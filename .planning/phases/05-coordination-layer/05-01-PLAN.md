    - from pathlib import Path
    - import aiofiles, uuid, magic
    - UPLOAD_DIR = Path("uploads")
    - MAX_FILE_SIZE = 100 * 1024 * 1024 (100MB)
    - ALLOWED_MIME_TYPES set containing: "image/jpeg", "image/png", "image/gif", "image/webp", "application/pdf", "application/vnd.openxmlformats-officedocument.wordprocessingml.document", "application/vnd.openxmlformats-officedocument.spreadsheetml.sheet", "video/mp4", "video/quicktime", "text/plain"
    - def get_upload_path(tenant_id: str, job_id: str, file_id: str, extension: str) -> Path — returns UPLOAD_DIR / tenant_id / job_id / f"{file_id}{extension}"
    - async def save_upload(tenant_id: str, job_id: str, file: UploadFile, max_size: int = MAX_FILE_SIZE) -> tuple[Path, str, int] — reads content, checks size, detects MIME with python-magic, generates UUID filename preserving extension, creates directory, writes with aiofiles, returns (storage_path, mime_type, file_size). Raises ValueError on invalid MIME or too large.
    - async def delete_file(storage_path: Path) -> None — unlinks file if exists

    **backend/app/core/websocket_manager.py:**
    - from fastapi import WebSocket
    - class ConnectionManager with:
      - self.connections: dict[str, WebSocket] = {} (user_id -> WebSocket)
      - self.subscriptions: dict[str, set[str]] = {} (user_id -> set of job_ids)
      - async def connect(self, user_id: str, websocket: WebSocket) — accept(), store connection, init empty subscription set
      - def disconnect(self, user_id: str) — pop from connections and subscriptions
      - async def subscribe(self, user_id: str, job_id: str) — add job_id to user's subscription set
      - async def unsubscribe(self, user_id: str, job_id: str) — discard job_id from user's subscription set
      - async def broadcast_to_job(self, job_id: str, message: dict) — iterate subscriptions, send_json to matching users, collect disconnected users and clean up
    - Global singleton: manager = ConnectionManager()
