**backend/app/api/v1/crew.py:**

Create router: `router = APIRouter(prefix="/api/v1/crew", tags=["crew"])`

Endpoints (following jobs.py patterns exactly for dependency injection, error handling, response models):

1. `POST /` — Create crew profile (admin only)
   - Accepts `CrewProfileCreate`, requires `require_admin` + `get_current_tenant` + `get_db`
   - Validates `user_id` exists and has `role=CREW`
   - Validates no existing CrewProfile for that user_id (return 409 if exists)
   - Creates CrewProfile with `tenant_id=tenant_id`
   - Returns 201 with `CrewProfileResponse`

2. `GET /` — List/search crew directory (any authenticated user)
   - Query params: `search: str | None`, `role: str | None = None`, `skills: list[str] | None = Query(None)`, `include_archived: bool = False`
   - Requires `get_current_tenant` + `get_db` (RLS handles tenant filtering)
   - Default: `WHERE archived_at IS NULL`
   - Search: ILIKE on User.email (join), CrewProfile.bio, CrewProfile.phone (using `or_` pattern from jobs.py)
   - Role filter: if `role` provided, join CrewAssignment and filter `WHERE CrewAssignment.role.ilike(f"%{role}%")` to find crew who have been assigned that role on any job. This enables filtering by functional role (e.g., "Camera Operator", "Sound Tech")
   - Skills filter: `WHERE CrewProfile.skills.contains([skill])` for each skill in list (AND logic — must have ALL)
   - Returns `list[CrewProfileResponse]`

3. `GET /{crew_id}` — Get single crew profile
   - Requires `get_current_tenant` + `get_db`
   - Returns 404 if not found

4. `PATCH /{crew_id}` — Update crew profile (admin only)
   - Accepts `CrewProfileUpdate`, uses `model_dump(exclude_unset=True)` pattern from jobs.py
   - Returns updated `CrewProfileResponse`

5. `POST /{crew_id}/archive` — Archive crew profile (admin only)
   - Sets `archived_at = datetime.now(timezone.utc)`
   - Returns 200 with updated profile

6. `POST /{crew_id}/unarchive` — Restore archived crew profile (admin only)
   - Sets `archived_at = None`
   - Returns 200 with updated profile
