    **GET /dashboard** (require_active dependency):
    - Look up CrewProfile where CrewProfile.user_id == current_user.id. If not found, raise 404 "Crew profile not found".
    - Query upcoming: SELECT CrewAssignment JOIN Job WHERE crew_id = profile.id AND (Job.scheduled_start >= now() OR Job.scheduled_start IS NULL) AND CrewAssignment.status != DECLINED. Order by Job.scheduled_start ASC NULLS LAST.
    - Query recent: SELECT CrewAssignment JOIN Job WHERE crew_id = profile.id AND Job.state == "complete" AND Job.scheduled_start >= (now() - 7 days). Order by Job.scheduled_start DESC.
    - Compute notification counts: reuse the exact logic from notifications.py get_notification_counts (inline it, don't import the endpoint). Query MessageLastSeen for unread message count, query CrewAssignment with status=PENDING for pending count.
    - Return PortalDashboardResponse with upcoming, recent, counts sections.
